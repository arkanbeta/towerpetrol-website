# Tower Petrol Website

Static multilingual website for **Tower Petrol General Trading Ltd.**

## Languages

- Arabic: default, RTL
- Kurdish Sorani: RTL
- English: LTR

## Tech stack

- HTML
- CSS
- JavaScript
- No backend
- No database
- No contact form

## Project structure

```text
towerpetrol-website/
├── index.html
├── css/
│   └── styles.css
├── js/
│   ├── translations.js
│   └── main.js
├── assets/
│   ├── logo/
│   └── images/
└── docs/
    └── maintenance-guide.md
```

## How to preview locally

Open `index.html` directly in your browser, or use VS Code with the **Live Server** extension.

## Deployment

This project can be deployed using GitHub Pages from the root folder of the `main` branch.
